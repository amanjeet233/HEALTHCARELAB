## 🔄 Integration Flow

```
1. User enters code
   ↓
2. Frontend validates (optional)
   ↓
3. Calls /api/promo-codes/validate
   ↓
4. Backend validates
   ↓
5. Returns validation result
   ↓
6. User clicks "Apply"
   ↓
7. Calls /api/promo-codes/apply
   ↓
8. Backend applies & increments counter
   ↓
9. Returns discount details
   ↓
10. Frontend updates cart display
   ↓
11. User proceeds to checkout
```
